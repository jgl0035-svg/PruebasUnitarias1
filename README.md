# Tarea Evaluación Módulo 14
## Prueba de Tests
### En este proyecto he trabajado con **JUnit 5** para crear pruebas unitarias y después implementar los métodos necesarios para que todas ellas pasen correctamente. El objetivo era aprender a validar datos típicos de un registro (nombre, contraseña, email y edad) siguiendo un flujo profesional: **primero los tests, luego el código.**
### El proyecto está dividido en dos partes principales:

### /main/java → contiene la clase ValidadorRegistro, donde implemento las validaciones.

### /test/java → contiene la clase ValidadorRegistroTest, donde escribo los tests con JUnit 5.
